Базовые элементы для Audi


Перед началом работы нужно:

  • - переместить нужные шрифты из папки '/resources/sass/core/audi_pack/fonts'
  • - в файле _configs.scss необходимо раскоментировать подключение шрифтов и нужные переменые для них
  • - в файле _core.scss нужно раскоментировать подключение одной из библиотек
  • - не нужные шрифты удаляем
  • - запускаем gulp


Кнопки


button - стандартная кнопка c классом .btn, для применения на сайте используем: @extend .btn

  • Кнопка содержит стили для сосотояний :hover, :active, :focus;
  • Шрифт AudiExtended
  • выравнивание по центру
  • Стандартные падинги 10px


button - кнопка серого цвета c классом .btn-dec, наследует все стили класса .btn; для применения на сайте используем: @extend .btn-dec;

button - кнопка красного цвета c классом .btn-inverse, наследует все стили класса .btn; для применения на сайте используем: @extend .btn-inverse;



button - кнопка с инлайновой иконкой c классом .with-ic-inline, для применения на сайте используем: @extend .btn;
@extend .with-ic-inline;

Саму иконку вставляем в псевдоэлемент :after

&:after{
@extend .s-icon-name;
}

button - кнопка содержит иконку с классом .btn-ic, спозиционированую абсолютом справа. Для блока с иконкой задаем:
@extend .btn-ic;
@extend .s-icon-name;

а для родительского элемента задаем:

position:relative;
padding-right: ..px;


Формы


Все input имеют обертку и этой обертке задается ширина и выравнивание относительно других элементов





Input с иконкой


Иконка расположена абсолютом в label.inp-with-ic-lbl имеет класс.inp-ic

Обертке задаем класс .inp-with-ic

Иконка наследует следующие классы

{
@extend .inp-ic;
@extend .s-icon-name;
}


Для чекбокса класс обертки .wrap-checkbox

Необходимо задать иконку для активного чекбокса

input:checked+label>.checked-ic:before{
@extend: .s-checkbox-ic;
}
